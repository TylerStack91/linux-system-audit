## Project Goal

The goal of this project is to practice basic Linux administration and security auditing commands. This project checks system information, users, network settings, running services, disk usage, and basic security details.

## Skills Practiced

-Linux command line  
-System information gathering
-User and permisssion review
-Process and service checking
-Basic cybersecurity documentation
